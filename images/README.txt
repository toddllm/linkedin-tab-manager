# Screenshots for LinkedIn Tab Manager

To add a screenshot of the extension pinned to the toolbar:

1. Take a screenshot of your Chrome extensions page showing the LinkedIn Tab Manager with "Pin to toolbar" enabled
2. Save the screenshot as "extension-toolbar-pin.png" in this folder
3. Ensure the image is a proper PNG file (not JSON or other format)
4. Update the README.md to reference this image if needed

## Example Extensions Page Screenshot

When taking a screenshot of the extensions page, capture:
- The "Pin to toolbar" toggle set to ON
- The extension name "LinkedIn Tab Manager"
- Any relevant permissions

## Example Toolbar Screenshot

When taking a screenshot of the Chrome toolbar:
- Make sure the LinkedIn Tab Manager icon is visible
- Include enough context to show it's in the Chrome toolbar
- Consider showing the popup when clicking the icon
